Put all your screenshots and images in THIS folder (docs/assets/).

The template pages already point to these filenames — just drop in images
with matching names and they'll appear automatically:

  logo.png
  getting-started-01.png, getting-started-02.png, getting-started-03.png

Tips:
- PNG or JPG both work. Keep widths around 1000-1600px for crisp display.
- To resize an image in the docs, use:  ![alt](assets/step-01.png){ width="600" }
- Use descriptive alt text in the brackets for accessibility.
